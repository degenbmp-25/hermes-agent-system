## Purpose

Create content systems that answer the subquestions AI search engines fan out from a broad buyer query.

## Procedure

1. Start with one buyer question or commercial topic.
2. Break the query into subtopics: problem, alternatives, criteria, pricing, risks, implementation, comparisons, and proof.
3. Map existing pages to subtopics and identify gaps.
4. For each gap, define the page type: guide, comparison, checklist, case study, glossary, integration, or FAQ.
5. Write briefs that answer the specific subquery and link back to the primary commercial page.
6. Add evidence: examples, screenshots, data, customer proof, or first-hand experience.
7. Track coverage by journey stage rather than one broad keyword only.

## Outputs

- Query fan-out map
- Content gap list
- Page briefs
- Internal linking plan

## Validation

- Each page has a specific subquestion.
- Content supports a buyer decision, not just keyword volume.
- Internal links connect supporting pages to the core offer page.
